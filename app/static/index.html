ABOUT

What

CDvist is a sequence-based protein domain search tool. It combines several popular algorithms to provide the best possible domain coverage for multi-domain proteins delivering speed-up, accuracy, and batch querying with novel visualization features.

Why

In the world of bioinformatics, combination of several tools often provides a better outcome than any of the tools alone.

How

The algorithm operates on subsequences rather than the entire sequence to cover domain information as completely as possible. It starts the search with the most specific tool and then fills the remaining gaps using more sensitive tools.
